About This Project

The Formula 1 Stats Explorer is a web-based application that allows users to explore driver standings, race results, and team performance data. This project is developed using Spring Boot as part of ISTE 240.

Main Features

  • View driver standings
  • Browse race results by season
  • Explore driver profiles
  • Compare team rankings
  • Admin management panel
